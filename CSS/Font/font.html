Understanding Font Styles in CSS

This page demonstrates different font styles and text formatting techniques using CSS.

1. Multiple Text Decorations

This text contains multiple decorations: underline, strikethrough, and overline.

2. Line Height

Line height controls the space between lines of text, improving readability.

3. Letter Spacing

Letter spacing increases or decreases the space between letters.

4. Text Indentation

Text indentation is used to move the first line of a paragraph inward.

5. Font Weights

Thin (100) - Used for a very light text appearance.

Light (300) - Slightly thicker than thin but still delicate.

Regular (400) - Default font weight.

Medium (500) - Slightly bolder than regular.

Bold (700) - Used for emphasis.

Black (900) - Heaviest and thickest font weight.

6. Font Style

This text is italicized using the `font-style: italic;` property.

7. Small Caps

Small caps format lowercase letters into uppercase with a smaller size.

8. Text Shadow

This text has a shadow effect, creating a 3D look.

9. Word Spacing

This text has increased word spacing to improve readability.

10. Text Alignment
